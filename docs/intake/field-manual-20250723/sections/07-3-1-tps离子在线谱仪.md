# 3.1 TPS离子在线谱仪

- Source chapter: 数据探测与采集模块
- Source section: 3.1 TPS离子在线谱仪

PyTPS解谱程序可以打靶时实时在线自动解谱、查看文件夹中已经存储的Andor照片离线解谱，以及可以在同一张图中绘制多张图片对应的谱线进行分析。

注意：使用打靶实时解谱功能时，需要在Andor相机控制程序所在的电脑上打开PyTPS解谱程序，PyTPS程序位置在电脑桌面的位置如下，使用时往往需要同时打开Andor相机及PyTPS解谱程序。

(1)安道尔相机的初始化设置；

在同一台电脑同时打开Andor相机控制程序，并检查设置。

- 打开Andor相机的电源，打开桌面上的“Andor SOLIS 64-bit”软件。

![3.1 TPS离子在线谱仪 image 65](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-065.png)

- 点击左下角的温度图标，开启冷却系统。

![3.1 TPS离子在线谱仪 image 66](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-066.jpeg)

- Cooler设置为On

![3.1 TPS离子在线谱仪 image 67](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-067.jpeg)

- 检查Andor相机设置，点击菜单栏的“设置”符号按钮

![3.1 TPS离子在线谱仪 image 68](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-068.jpeg)

- 检查Camera Setup选项是否与下图一致，尤其是设置触发Triggering选择External

![3.1 TPS离子在线谱仪 image 69](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-069.jpeg)

- 检查Autosave选项卡，设置照片的保存路径，Start Value重新设置为1，检查Tiff Option中是否为16-bit Gray

![3.1 TPS离子在线谱仪 image 70](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-070.jpeg)

- 点击相机图标，Andor相机软件即设置完成，将其最小化后可以设置PyTPS软件，进入打靶实时自动解谱流程。

![3.1 TPS离子在线谱仪 image 71](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-071.jpeg)

(2)解谱程序的初始化设置；

![3.1 TPS离子在线谱仪 image 72](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-072.png)

- 首先打开PyTPS主程序的主界面，如下图：

![3.1 TPS离子在线谱仪 image 73](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-073.png)

- 读取安道尔相机存储图片的文件路径，在程序左上角“文件路径”处点击“浏览”，选择Andor相机图片保存的文件夹。

![3.1 TPS离子在线谱仪 image 74](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-074.png)

- 能谱保存路径设置：程序右下角“保存路径”处点击“浏览”，选择本程序解谱后得到的能谱曲线图片以及能谱曲线.txt文件等结果保存的文件夹。通常可以在Andor相机图片保存文件夹下新建名为“处理后”的文件夹来保存。

![3.1 TPS离子在线谱仪 image 75](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-075.png)

(3)在线模式实时打靶解谱基本流程；

- 点击PyTPS面板的在线模式，程序即进入在线自动解谱状态，按照安道尔相机的初始化设置步骤设置好相机触发后，Andor相机每触发一次，本程序就可以自动完成解谱。

![3.1 TPS离子在线谱仪 image 76](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-076.png)

- 当得到第一张有质子能谱线的照片后，应利用这张照片校正当前原点位置。校正方法如下：

![3.1 TPS离子在线谱仪 image 77](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-077.png)

- 选中“显示参考抛物线”，伪彩图中会出现一条淡蓝色的参考抛物线。结合淡蓝色参考抛物线，以及黑色的十字准星，调节X0 Y0（原点坐标）使得抛物线与质子打出的曲线中心重合。

- 注意：原点十字准星仅作为基本参考，最终的设置应该以抛物线为准。

- 调好后如下图所示，程序可以正常解谱。伪彩图和能谱图上黑色竖线就对应着程序自动得到的截止能量位置。

![3.1 TPS离子在线谱仪 image 78](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-078.png)

- 勾选“自动保存”，可以将右侧的能谱图以及能谱图对应的坐标.txt文件保存到上面选择的路径中。

![3.1 TPS离子在线谱仪 image 79](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-079.png)

(4)自动解谱功能说明；

- 图片缩放：点击“图片缩放”按钮后，在伪彩图界面上画一个框，可以放大选定区域的伪彩图（拖动过程中并不会有框显示出来，这是正常现象）。再次点击图片缩放可以进一步放大，点击“原始大小”即恢复。

- 伪彩图设置：伪彩图类型可以修改，但推荐使用默认的partical类型，更加直观清晰。范围上下限可以修改颜色映射的对应范围，上下限为0~65535。

- 能谱图设置：修改能谱线窗口中的坐标轴范围。

![3.1 TPS离子在线谱仪 image 80](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-080.png)

- ±dY（重要）：参考抛物线对应的是质子抛物线的中心，而±dY即为程序解谱时考虑的抛物线的宽度（单位为像素）。对于信号较弱的图片，可以调高这个值，但数值最大为11，更大则没有物理意义。过分调高这个值会导致能谱线的高能端出现更多噪声，一般建议这个值在3-8之间调节。

- 自动解谱窗口（重要）：自动解谱功能是对能谱线从高能向低能扫描，寻找从此开始连续的一段能量值区间内，能谱线始终高于本底噪声曲线的对应能量值，作为截止能量。因此，对于高能端噪声较大的情况，可以调大这个值。更小的窗口可以有利于分辨高能端的微弱信号，但更容易出现误判；更大的窗口则反之。一般这个值设定在0.5左右即可。

(5)手动解谱功能说明；

- 图片缩放：点击“图片缩放”按钮后，在伪彩图界面上画一个框，可以放大选定区域的伪彩图（拖动过程中并不会有框显示出来，这是正常现象）。再次点击图片缩放可以进一步放大，在能谱图（右侧）中点击对应的位置，黑色竖线光标可以手动移动到点击的位置，同时左侧伪彩图中的光标也会移动到对应的位置，同时将给出光标处对应能量。当自动解谱功能出现故障时，可以利用这一功能手动解谱并手动记录。

![3.1 TPS离子在线谱仪 image 81](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-081.png)

(6)离线模式及能谱分析模式；

- 离线模式：选择离线模式后，不会再继续自动触发Andor相机，左侧的图片列表可以点击，程序将完成对选中的图片的解谱工作。

![3.1 TPS离子在线谱仪 image 82](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-082.png)

- 能谱分析模式：勾选后将自动进入离线模式，左侧图片列表可以多选，伪彩图窗口将失效，程序将在右侧一个能谱图中绘制出选中的所有图片对应的谱线，可以对照参考。

![3.1 TPS离子在线谱仪 image 83](../images/all/FieldManual-20250723-3-1-tps离子在线谱仪-083.png)

- 程序退出时：覆盖保存当前设置，即保存目前窗口中的所有设置，包括文件路径、原点坐标等等。推荐只要完成了正确设置正常使用程序后，退出程序时，均选择覆盖保存。若误动了某些设置，可以选择不保存，重启后程序将恢复到上一次打开时的状态。
