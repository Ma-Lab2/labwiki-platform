## 1.3.1 模拟光的启动与光强调节

![1.3 模拟光系统 image 4](../images/all/FieldManual-20250723-1-3-模拟光系统-004.png)

*图注：图 1-4*

模拟光控制盒如图 1-4(a)，开启先按开关，再拧钥匙，显示数字后代表正常开启。正常情况下开启后为0.00，代表光强为0，每次关闭前也要将光强降为0；

连接品质提升腔主控电脑（10.0.23.109），如图 1-4(b)所示，运行python，打开smartlink，先运行commotor脚本，再运行Qcontrol脚本，会出现串口点击运行面板；

正常运行可得到模拟光电机转轮的控制面板，正方向为减光强，负方向为加光强，每次加减500为比较合适的步长。每次使用完将位置回到0，避免电机卡死。


## 1.3.2 模拟光的导入

仅观察模拟光在同轴远近场的指向（不经过PM）

打开IP10.0.23.108电脑 电机控制软件

点击 M1移出

点击 M2移出

点击 M3移出

需经PM调光

点击 M1移出

确认 M2、M3已移入

移出模拟光

点击 M1移入


## 模拟光的各种应用场景与常用强度数值记录

模拟光主要用来做打靶前的调试，常用情况为调节PM，做PM靶点规划，一般采用PM银镜面时，模拟光强度开到1.7左右比较清晰。在PM焦点聚焦时，由于PM银镜面反射率较高，一般采用PM镀膜面进行聚焦，此时模拟光光强可降到0.02左右，如果聚焦效果不理想，可适当调高，常用的模拟光强度如表 1-1。
