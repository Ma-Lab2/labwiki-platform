# 4.3 打靶前安全检查

- Source chapter: 工作模式与作业流程
- Source section: 4.3 打靶前安全检查

（1）第一发次前检查

至打靶腔全链路畅通（特别注意 PM 远场、 同轴远近场、取样镜、防溅射膜、防回光） 需拍照发送到工作群

验证腔为打靶腔时（主激光依次通过 M1 M2 PM M3 M5 M6）监控逐个检查镜子到位

主激光 OAP 姿态到位，等离子体镜系统离焦到位，OAP1 2到位

复查真空及高压安全 需拍照发送到群

tps 小孔打开

应衰减单元准备好（315、靶前、时间诊断平台）见附录

应触发设备准备好（315、靶前、谱仪、闪烁体、透射光斑）见附录

靶后位置安全，靶后相机关闭、tps 在工作位、RCF 安全位、防回光是否接入、溅射膜是否接入

最后确认一次清场，人员安全

询问激光是否准备好

（2）打靶途中

先切靶，再动 OAP

PM 到位报告、靶体到位报告、OAP 到位报告

击发（打靶）

（3）打靶途中二次查靶

- 同轴远近场到位遮挡漏光

- 靶后观察

- 观察结束后，靶后移开，靶后相机关闭，靶前衰减加上

- 同轴远近场移走

（4）打靶结束

- 等激光关闭后，切换打靶系统为调光模式，关闭 tps 小孔

- 下靶场关闭通光阀，降高压，最后放气

- 关闭应关设备，参考最后一章里的应关闭设备一览
