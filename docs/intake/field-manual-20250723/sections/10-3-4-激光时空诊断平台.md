# 3.4 激光时空诊断平台

- Source chapter: 数据探测与采集模块
- Source section: 3.4 激光时空诊断平台

- 激光时空诊断平台主要包括了315空间诊断平台、M7漏光空间诊断平台、DM腔时间诊断平台、PM腔透射光诊断平台。

3.4.1 315空间诊断平台

- 基准同轴远近场可以精确监控靶场光路的位置与指向，但缺点是只能在弱光下使用，而且会阻挡光路传输，无法监测打靶时激光的状态。于是在315工作台搭建了漏光缩束实时远近场，为了在打靶的同时实时监测当前激光的状态，尤其是光斑的形状。

![3.4 激光时空诊断平台 image 84](../images/all/FieldManual-20250723-3-4-激光时空诊断平台-084.png)

图：漏光缩束实时远近场布置示意

- 如下图所示，漏光缩束实时远近场是利用可变形镜后的反射镜 M4 的漏光，通过大口径石英窗片导出腔外搭建的。

![3.4 激光时空诊断平台 image 85](../images/all/FieldManual-20250723-3-4-激光时空诊断平台-085.jpeg)

- 由缩束镜头输出的略发散的光束，经过 f=150 mm 的缩束镜头重新准直后，传输到面包板上固定的远近场光路中。准直的光束先经过 f=300 mm 的透镜聚焦，在焦点前使用一片反射率4%的薄玻璃片分束，反射光作为远场，远场相机直接接收焦点。分束的透射光再经过 f=100 mm 的透镜准直，相比入射光 1：3 再次缩束后入射到近场相机中。光路中平行光部分两透镜间的距离经过调节，重新将成像面确定在可变形镜表面。

3.4.2 M7空间诊断平台

- 位于验证腔外，通过一块两英寸窗片将激光经过反射镜的透射光导出，腔内通过一块大透镜将光束缩束，M7工作平台上可以用于监测激光到靶时的状态，比如到靶能量以及光斑形状，也可以将三阶自相关仪放置于此测量激光对比度，作为实验时的激光实时诊断平台。

3.4.3 DM腔时间诊断平台（中帅补充软件控制）

- 激光时间诊断平台位于变形镜腔外，通过一块一英寸的窗片将主激光一部分取样出来；时间诊断平台上的设备可以分析导出来的激光，得到主激光的参数以及光束质量；如下图所示，导出的激光会从右侧窗片导出，通过爬低架导到下图电机的反射镜上，电机移动到不同位置，可以将激光导入不同的诊断设备中；电机上的反射镜在最右侧时，对应光学平台上写的“二阶”，此时激光导入二阶自相关仪中，可以用于测量激光的脉宽，诊断激光压缩质量；电机第二档为光谱测量，可以测量激光的光谱形状；电机第三档将激光导入FROG设备，用于测量激光的脉宽与相位；电机第四档将用于测量激光的偏振状态；电机第五档可以观察激光的远近场，用于校准激光的角度，确保其他设备正常接收；电机第六档将激光导入三阶自相关仪，用于测量激光的对比度。
