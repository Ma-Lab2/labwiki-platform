# 4.1 远近场同轴

- Source chapter: 工作模式与作业流程
- Source section: 4.1 远近场同轴

远近场同轴主要是为了检查激光每天的指向漂移量，并及时校准到合适的指向上，方便进行下一步实验。远近场同轴实验装置由一块掏了小孔的陶瓷板和远场指向光路组成。示意图如图所示，其中通过镜头直接观察陶瓷板屏幕上的光斑轮廓为指向中的近场监控，通过小孔传输到后端镜组最后进入到远场监控相机中的为远场监控，该装置只能在小于5mJ的激光等级下工作。

(1)操作流程

- 确保真空系统安全，打开通光阀以及各个腔的阀门；

- 打开108电脑，控制电机轴2，移动到395的位置；

- 打开迈德威视相机基准近场与基准远场；

- 若放模拟光，则M1移出，若放主激光，则M1移入；

- 模拟光强度开到1.7左右；

- 观察基准近场与远场相机上的光斑与标定的十字线有没有发生偏移。

(2)安全事项

- 确保所有镜子在位之后再打开激光；

- 若使用模拟光，确保PM在银镜位，否则光强可能不够；

- 若在打过的点位则可能散射打坏相机；

- 若使用主激光，根据激光强度考虑使用石英面、银镜面或镀膜面。
