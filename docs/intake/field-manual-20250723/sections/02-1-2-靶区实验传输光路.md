如图 1-3所示，激光沿红线传输，经 M1–M7 与 OAP，依次穿过爬低腔、品质提升腔、PM更换腔、变形镜腔、综合实验腔、验证腔；主加速腔在束线区（未展示在图中）。

- 可移动反射镜及其功能：

- M1：主激光 / 模拟光切换

- M2、M3：PM 有无切换

- M5：光路是否到综合实验腔

- M6：光路是否到主加速腔

控制软件在品质腔副控电脑（10.0.23.108）。
