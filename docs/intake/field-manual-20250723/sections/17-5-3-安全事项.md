# 5.3 安全事项

- Source chapter: 安全工作与规范工作要求
- Source section: 5.3 安全事项

5.3.1用光安全

- （1）模拟光

- 注意放光前各个镜子是否在位

- 调光全过程关闭通光阀，禁止主激光进入后续腔体

- 功率尽量不开到2以上。

- （2）主激光

- 开光前广播并口头确认靶场全部人员已知情；

- 确认各个腔体窗片的挡光帘在位；

- 弱光调试时，所有人佩戴护目镜，背对腔体；

- 强光调光与打靶时，所有人远程操作，靶场不能有人；

- 若有人不慎被激光所伤，停止实验，立刻向安全负责人上报。

5.3.2用电安全

- （1）高压电源

- 操作时须佩戴高压防护手套；

- 必须两人在场，一人为操作员，另外一人为安全员

- （2）走线

- 所有布置的走线要有所标记；

- 插排不可乱拉，大功率电器需做特殊标记。

5.3.3重要设备安全

- （1）真空腔体内

- 注意不要碰到反射镜、OAP、变形镜等镜面，避免刮花镜面；

- 不要留与打靶无关的设备、工具在腔内

- 进腔的新装备要工程会报备

- （2）真空腔体外

- 走路注意安全，避免磕碰光学平台、通光通道等设备；

- 强磁单独存放，大型设备需有两人以上在场时才可以挪动。
