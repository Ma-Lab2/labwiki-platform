# 4.2 波前矫正与验证腔聚焦

- Source chapter: 工作模式与作业流程
- Source section: 4.2 波前矫正与验证腔聚焦

4.2.1 波前矫正

（1）激光传送到综合腔

- 根据主激光传送到综合腔要求检查所有腔体阀门和镜子到位；

- 检查综合腔OAP工作位

- 检查综合腔靶后工作位

- 确定激光能量为1mJ档（由操作人员现场研判，需要大于1mJ要报告）

（2）综合腔对准以及聚焦优化

- 操作员调节综合腔OAP，看到主激光后将所有设备调节为触发采集模式；

- 继续调节综合腔OAP，将焦斑优化到当前DM面型下的最佳水平；

![4.2 波前矫正与验证腔聚焦 image 89](../images/all/FieldManual-20250723-4-2-波前矫正与验证腔聚焦-089.png)

（3）129个触动器矩阵响应校准

- 打开SID4软件，此处操作可参考1.7节；

- 进入矫正单元，开始矫正，根据需要进行粗矫正和重新矫正

- 矫正完成后，检查矫正效果

（4）波前优化，数据存储

- 选择波前优化的矩阵模式；

- 选择波前优化的方案，一般选择闭环模式，选择按照次数优化

- 开始波前矫正，矫正完成后，保存面型。

4.2.2 验证腔聚焦

（1）激光传送到验证腔

- 根据主激光传送到验证腔要求检查所有腔体阀门和镜子到位；

- 检查验证腔OAP工作位

- 检查验证腔靶后工作位

- 确定主激光工作能量为68nJ档

（2）聚焦和束靶耦合

- 打开主激光

- 低倍镜下看到光斑

- 根据需要调节光强，通过切换PM不同材料工作位实现

- 优化OAP实现最佳聚焦

- 关闭主激光，打开找靶灯

- 以靶后为参考，移动靶架观察针尖，让针尖清晰，完成束靶耦合
