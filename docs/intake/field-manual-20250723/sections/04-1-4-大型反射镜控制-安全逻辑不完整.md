1.6.2调光状态等离子体镜

- 等离子镜中间为镀膜面，极低反射率；左侧未镀膜，为石英面，约为4%反射率；右侧镀铝膜，一般称为银镜面，反射率接近100%；

1.6.3打靶状态等离子体镜

- 手动打靶：

- 如下图所示，无论是手动打靶还是自动打靶，都是在touchview的自动界面进行操作。手动打靶不需要点击别的按钮，只需要根据excel提前列好的每个靶点位置对点击的xyz三个轴进行移动即可；

- 由于斜打靶的回光特性，等离子镜触发后会在其坑位右侧有一个小的回光坑，所以为了避免已打过的等离子镜点位对未打过的点位造成影响，一般整块镜子从右往左开始打，等离子镜出发后的坑位如下图所示。

1.6.4等离子体镜的更换与维护

- 更换等离子镜后，虽然电机有限位功能，但是也会造成微弧度级别的误差，对应到点位的偏移，以及前后离焦，可能也会带来百微米级的误差，为了保证换等离子镜前后位置不变，因此需要重新校准；

- 校准时需要挪动电机，将等离子镜切回银镜面，去改变旋转俯仰改变离焦，使同轴远近场和之前的位置一样，保证不能卡光；

- 注意只移动等离子镜电机，不改变OAP位置；

- 在动等离子镜电机的旋转和俯仰时，近场改变会比较敏感，动z轴离焦时，远场改变会比较敏感；

- 一般情况下，校准时等离子镜电机x轴是不动的，一般是动z轴和旋转俯仰来找回去，找完之后，将数据写进自动打靶数据中去。若是手动打靶，同样把找好的银镜面、石英面数据更新进excel表格中去；

- 需要注意的是，切换不同等离子镜时，俯仰的回程差是比较小的，基本没有；但是电机的旋转齿轮比较大，精确度较低，因此有时会造成较大回程差，所以只要旋转了等离子镜的面，都需要重新校准；

- 另外，弱光使用的石英面和银镜面不能出现在excel表格里，因为这两个面无法消除回光，在使用大角度打靶之前，不可以应用在打靶里；

- 对于自动打靶，如果一次实验需要扫描不同的离焦，可以提前在excel表格中进行设置。

1.7.1使用流程

导光与成像：

![1.4 大型反射镜控制（安全逻辑不完整） image 26](../images/all/FieldManual-20250723-1-4-大型反射镜控制-安全逻辑不完整-026.png)

成像效果

- 协调开光人员，控制合适的能量激光输送到综合腔，一般控制在1mJ以下，LOOP成像系统的低倍相机、高倍相机、近场相机全部打开，综合腔靶前相机打开、SID4的相机在另外一台电脑上同步打开

- 所有相机打开后，将系统的衰减级别加至最高，把系统移入工作位，sid4先采用连续曝光模式，这样能够保证看到激光时不过饱和

- 在sid4的视野内看到激光并且不饱和后，调整衰减级别，使得sid4的光强曝光值控制在2000左右，显示绿条即为工作曝光区间

- 此时可以打开所有相机的触发采集功能和sid4的触发采集功能，让激光输出进入连续触发模式

- 触发模式能够正常采集到激光后，调整成像系统（相当于用OAP聚焦综合腔靶点，主要消除指向偏差带来的像散和离焦）

- 调节到比较好的状态后，进入下一步

触发与mask选择：

- 进入相机的控制界面，选择mask的大小，这里大小要控制在190个像素内，太大了系统没有办法启动

![1.4 大型反射镜控制（安全逻辑不完整） image 27](../images/all/FieldManual-20250723-1-4-大型反射镜控制-安全逻辑不完整-027.png)

校准：

- 关闭相机界面，进入校准模块，系统会提示选择校准模式，第一个模式是在光路已经发生改变的情况下做校准，第二个模式是只改变mask大小的模式下做校准

- 点击校准开始就可以开始进行响应矩阵校准，要注意观察校准过程中变形镜和sid4波前探测是否都正常，二者均正常才是正确的校准流程

![1.4 大型反射镜控制（安全逻辑不完整） image 28](../images/all/FieldManual-20250723-1-4-大型反射镜控制-安全逻辑不完整-028.png)

- 如果发现校准电机全部变红，则右键勾选取消掉红色块以后继续进行校准，红色块的意思此电机不进行校准

闭环矫正：

- 校准完成后进入LOOP模块，在livephase选项卡中点击倾斜项参考，这样在校准的时候就不会进行倾斜项的矫正，LOOP有三不校，离焦项不校，倾斜两项不校，也就是泽尼克系数中的前三项

- 校准完成后进入LOOP模块，选择迭代次数和迭代效率，点击closed-loop进行闭环校准

![1.4 大型反射镜控制（安全逻辑不完整） image 29](../images/all/FieldManual-20250723-1-4-大型反射镜控制-安全逻辑不完整-029.png)

- 有两种闭环模式，一种是按照rms阈值，一种是按照迭代次数，一般选择迭代次数，可控性更高
